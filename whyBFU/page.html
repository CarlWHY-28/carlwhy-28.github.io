Driver Fatigue Detection Algorithm (Graduate Project)

March 2024 - current


• Driver Fatigue Detection Algorithm, is an algorithm to evaluate driver status.
• It is an analysis algorithm based on detection and pose estimation. YOLOv8-pose is adopted as a pose estimation method, and facial keypoint detection is realized by using functions in Dlib.
• Deep learning, posture analysis, facial keypoint analysis, sliding window and etc.

RCT_MLPDocAnalyzer

March 2024


• RCT_MLPDocAnalyzer is a document analysis tool to classify whether a paper uses randomized controlled trial based on some of the paper's properties from the document. 
• The project uses CountVectorizer in conjunction with neural networks to analyze text.
• Machine Learning, feature Engineering and etc.
